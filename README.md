# Craddock Connection
| Craddock family website

### Dependencies
* node (v16+)
* npm
* Astrojs
* Reactjs
* Tailwindcss

## Local Development
* node v16+
 - [nvm](https://github.com/nvm-sh/nvm) allows you to switch between node and npm version
* [astrojs](https://docs.astro.build/en/install/auto/)
 - `npm install astro`
* run craddock connection locally
 - `npm install`
 - `npm run dev`
